# scratch-my-own-itch

Personal tools built to solve my own problems. Not tutorials, not demos — just things I needed and built.

---

### [expense-tracker](./expense-tracker/)
A personal life dashboard — expense tracking, budgeting, recurring transactions. Next.js + FastAPI + SQLite. Built because I wanted something that fits exactly how I think about my money.

`Next.js` · `TypeScript` · `FastAPI` · `SQLite`

---

### [substack-archiver](./substack-archiver/)
Captures paywalled Substack articles as PDFs and indexes them into a vector database. Searchable via Claude Code through an MCP server. Built because I kept losing track of newsletters I paid for.

`Python` · `Playwright` · `ChromaDB` · `FastMCP` · `Docker`

---

### [banana-costume-generator](./banana-costume-generator/)
Streamlit app that generates custom banana costume images using Gemini's image generation API. Built for fun. No further justification needed.

`Python` · `Streamlit` · `Gemini API`

---

### [property-finder](./property-finder/) _(2017, archived)_
Scraped property listings from iproperty.com.my, filtered by budget and area, and plotted them on an interactive map. Built to help me find and buy my house in KL. It worked. Not maintained — iproperty.com.my has changed significantly since 2017.

`Python` · `Scrapy` · `Flask` · `Folium` · `Pandas`
